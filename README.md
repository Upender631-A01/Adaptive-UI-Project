# Adaptive UI using AI/HCI Project

## What this is
A simple adaptive dashboard that changes its UI at runtime based on user behavior.

## Adaptive behaviors
- Reorders menu items based on click frequency
- Highlights the most-used menu item
- Highlights the most-used category
- Updates recommendations based on user preferences
- Stores a user profile locally using localStorage

## AI logic used
Rule-based decision logic (symbolic AI):
- If an item reaches 3+ clicks, it is prioritized and highlighted.

## How to run
1. Open `index.html` in a browser (Chrome/Edge).
2. Click menu items and categories.
3. Refresh to see the personalization persist.
